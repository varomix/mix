Summary

Optimizations Applied

  • -O2 on runtime.c — The cc link step now compiles the MIX runtime (list, map, string, shared, task functions) with -O2, bringing runtime-heavy benchmarks to near-C speed.
  • Compile-time constant folding — Binary operations on literal ints/floats (+, -, *, /, %) are evaluated at compile time, emitting a single copy instead of two loads + an operation.
  • Direct literal assignment — Immutable variables initialized with literals skip the intermediate temp register and emit the value directly.
  • Self-pointer elision — Method field access uses %v.self directly in load/add instructions instead of copying to a fresh temp.
  • Const memoization — @const values are emitted once per function and the temp is reused on subsequent references.
